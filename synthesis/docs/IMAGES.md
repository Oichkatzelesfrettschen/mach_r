# Images and Artifacts

_Contributing: see [AGENTS.md](../../AGENTS.md) for guidelines._

Generated by `xtask` under `build/`:

- `build/dist/`
  - `mach_r_kernel.elf` — kernel ELF (AArch64).
  - `mach_r_kernel.bin` — binary image (via objcopy when available).
  - `SHA256SUMS` — checksums for ELF/BIN and created images.

- `build/images/`
  - `mach_r.qcow2` — compressed disk image (from raw).
  - `mach_r.iso` — ISO with `boot/kernel.bin` and `README.txt`.

Commands
- `cargo run -p xtask -- filesystem` — create base sysroot for future packaging.
- `cargo run -p xtask -- disk-image` — create raw and qcow2 images.
- `cargo run -p xtask -- iso-image` — create ISO; appends checksum to `SHA256SUMS`.

Notes
- On macOS, ISO uses `hdiutil`; on Linux, `genisoimage` must be installed.
- Checksums are appended to `build/dist/SHA256SUMS`. Upload along with images.
